A General Toolkit for Online Learning Approaches

Comprehensive collection of state-of-the-art algorithms for online learning, active learning, and concept drift adaptation. Designed for researchers and practitioners working with dynamic data streams.

View on GitHub

Key Features

Explore the powerful capabilities of our online learning toolkit designed for dynamic data environments

1

Online Active Learning

Comprehensive toolkit featuring state-of-the-art OAL strategies designed for dynamic data streams.

  • CogDQS: Cognitive Dynamic Query Strategy
  • DSA-AI: Dynamic Sample Allocation with Active Inference
  • MTSGQS: Multi-Task Selective Global Query Strategy
2

Drift Adaptation

Advanced classifiers specifically designed to handle concept drift in imbalanced and evolving data streams.

  • ROALE-DI: Robust Online Active Learning Ensemble for Drift and Imbalance
  • OLI2DS: Online Learning for Imbalanced and Drifting Streams
  • Adaptive windowing techniques
3

Semi-Supervised Learning

Specialized OSSL classifiers that effectively utilize unlabeled data in streaming environments.

  • OSSBLS: Online Semi-Supervised Broad Learning System
  • ISSBLS: Incremental Semi-Supervised Broad Learning System
  • Self-training and co-training strategies
4

Ensemble Methods

Powerful ensemble techniques that combine multiple learners for robust performance in dynamic conditions.

  • ARF: Adaptive Random Forest
  • DES: Dynamic Ensemble Selection
  • Online bagging and boosting
5

Visualization Tools

Interactive visualization capabilities for intuitive performance comparison and analysis.

  • Dynamic GIFs of model performance
  • Interactive confusion matrices
  • Real-time performance metrics
6

Research-Oriented

Developed by THUFDD Research Group with implementations of cutting-edge algorithms published in top-tier journals.

  • IEEE Transactions on Knowledge and Data Engineering
  • Pattern Recognition
  • Neural Networks

Real-time Model Comparison

Our visualization tools allow you to compare the performance of different models in real-time as they adapt to concept drift in data streams. The dynamic GIF demonstrates how various algorithms respond to changes in the underlying data distribution.

Key metrics such as accuracy, F1-score, MSE, and R² are tracked and visualized to help researchers and practitioners understand model behavior under different conditions.
